Package pyrdnap

RD NAP 2018 conversions between geodetic GRS80 (ETRS89) lat-/longitudes and heights and Netherlands' local RijksDriehoeksmeeting (RD) coordinates and Normaal Amsterdams Peil (NAP) quasi-geoid-heights.

Note: PyRDNAP and pyrdnap have not been formally validated and are not certified to carry the trademark name RDNAPTRANS(tm).

See Also: Module pyrdnap.rdnap2018 for further information.

Version: 26.05.05

Submodules

Function Details

validation3 (self_txt, R, all_=False, in_out=True, _print=None, _printest=None)

 

Run the official RDNAP 2018 self-validation tests.

Arguments:
  • self_txt - Name of the file containing the RDNAP 2018 self-validation tests (str), .../RDNAPTRANS2018_v220627/.../Z001_ETRS89andRDNAP.txt.
  • R - An RDNAP2018v# transformer (RDNAP2018v1 or RDNAP2018v2 instance).
  • all_ - If True print all tests and test results, otherwise only failing tests (bool).
  • in_out - If True test only points inside the RD region, if False only points outside (bool).
  • _print - A Python 3+ print-like callable or None to not print the header and final, summary lines.
  • _printest - A Python 3+ print-like callable or None to not print any lines for any test or failing test.
Returns:
3-Tuple (failed, total, in_outside) with the number of FAILED tests, the total number of tests and the number of test lines in- or outside the RD region.

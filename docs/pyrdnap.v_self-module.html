Function validation3 to run the RD NAP 2018 self-validation tests .../RDNAPTRANS2018_v220627/.../Z001_ETRS89andRDNAP.txt obtainable from NSGI.NL after registration.

For each test line, 3 lines are produced: the 1st showing the point id, the original (ETRS89) lat, lon and height and the expected RDx, RDy and NAPh values.

The 2nd line shows the lat, lon and height and RDx, RDy and {NAPh} results from the RDNAP2018v1 or -v2 transformer's reverse respectively forward method.

The 3rd line contains the (absolute value) of the differences between the results on the 2nd line and the corresponding, original value on the 1st line.

The final 2 lines of the output are the maximum of all (absolute value) differences followed by a line with the formal RDNAP 2018 requirements, 0.000000010 degrees or 0.0010 meter for each result.

Notes:
  • A test FAILED if any reverse or forward result exceeds the RD NAP 2018 requirement for that result.
  • For tests with NAPh marked "*", only the reverse lat and lon and forward RDx and RDy results are taken into account.

See Also: Module pyrdnap for examples to invoke validation3.

Function Details

validation3 (self_txt, R, all_=False, in_out=True, _print=None, _printest=None)

 

Run the official RDNAP 2018 self-validation tests.

Arguments:
  • self_txt - Name of the file containing the RDNAP 2018 self-validation tests (str), .../RDNAPTRANS2018_v220627/.../Z001_ETRS89andRDNAP.txt.
  • R - An RDNAP2018v# transformer (RDNAP2018v1 or RDNAP2018v2 instance).
  • all_ - If True print all tests and test results, otherwise only failing tests (bool).
  • in_out - If True test only points inside the RD region, if False only points outside (bool).
  • _print - A Python 3+ print-like callable or None to not print the header and final, summary lines.
  • _printest - A Python 3+ print-like callable or None to not print any lines for any test or failing test.
Returns:
3-Tuple (failed, total, in_outside) with the number of FAILED tests, the total number of tests and the number of test lines in- or outside the RD region.

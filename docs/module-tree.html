Module Hierarchy

  • pyrdnap: RD NAP 2018 conversions between geodetic GRS80 (ETRS89) lat-/longitudes and heights and Netherlands' local RijksDriehoeksmeeting (RD) coordinates and Normaal Amsterdams Peil (NAP) quasi-geoid-heights.
    • pyrdnap.__main__: Run a RD NAP 2018 forward or reverse conversion round-trip examples or run the entire RD NAP 2018 self-validation test suite in various ways.
    • pyrdnap.__pygeodesy: (INTERNAL) pyrdnap access to some private pygeodesy attributes.
    • pyrdnap.rd0: (INTERNAL) RD and RD0 constants and RDNAP7Tuple and RDregion classes.
    • pyrdnap.rdnap2018: A pure Python implementation of the Netherlands' RD NAP specifications to convert between geodetic GRS80 (ETRS89) lat-/longitudes in degrees and local RijksDriehoeksmeeting (RD) coordinates and Normaal Amsterdams Peil (NAP) quasi-geoid-height in meter.
    • pyrdnap.v1grid: Variant 1 _NAP_h, _lat_corr and _lon_corr grids.
      • pyrdnap.v1grid.nlgeo2018_txt_NAP_h
      • pyrdnap.v1grid.rdcorr2018_txt_lat_corr
      • pyrdnap.v1grid.rdcorr2018_txt_lon_corr
    • pyrdnap.v2grid: Variant 2 _NAP_h, _lat_corr and _lon_corr grids.
      • pyrdnap.v2grid.naptrans2018_txt_NAP_h
      • pyrdnap.v2grid.rdtrans2018_txt_lat_corr
      • pyrdnap.v2grid.rdtrans2018_txt_lon_corr
    • pyrdnap.v_grids: (INTERNAL) RD NAP 2018 v#grid utilities.
    • pyrdnap.v_self: Function validation3 to run the RD NAP 2018 self-validation tests .../RDNAPTRANS2018_v220627/.../Z001_ETRS89andRDNAP.txt obtainable from NSGI.NL after registration.
